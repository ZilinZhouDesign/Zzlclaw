# 观鸟记录网站

一个简单美观的个人观鸟记录管理网站，支持添加、删除观鸟记录，自动加载鸟类图片，数据本地存储。

## 功能特性

- ✨ 美观的渐变色界面，响应式设计
- 📝 快速添加鸟类名称和备注
- 🖼️ 自动加载对应鸟类的实拍图片
- 📊 实时统计记录总数、本月新增和独特鸟类数量
- 💾 数据保存在浏览器本地，无需后端
- 📱 完美适配移动端和桌面端

## 本地运行

```bash
# 方法1：直接用浏览器打开 index.html
open index.html

# 方法2：使用本地服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 使用说明

1. 在输入框中填写鸟类名称和备注信息
2. 点击"添加记录"按钮或按回车键
3. 系统会自动加载该鸟类的图片并创建卡片
4. 点击卡片上的"删除"按钮可以移除记录
5. 所有数据自动保存在浏览器中，刷新页面不会丢失
